About Footylytic

Footylytic is a web application written entirely in python that uses historical (statistical) data as well as data taken from Twitter to determine the outcome of various football matches.

Future games (games that are not being played today) are more heavily weighted towards using the statistical data provided whereas any game in progress will have more tweets about it on the day and therefore takes more emphasis on it.

The analysis provided only covers the premier league and from that, only a select few teams due to the differences in playing style. Currently the system only supports: Arsenal, Manchester United, Watford, Everton, Newcastle and Stoke City.
